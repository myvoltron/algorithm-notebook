# Coding Test Practice

파이썬으로 코딩 테스트 문제를 풀고, 자주 쓰는 알고리즘 패턴을 정리하는 개인 연습 저장소입니다.

문제 풀이 파일은 출처나 난이도, 주제별 디렉터리에 나누어 보관합니다. 풀이를 다시 읽을 때 빠르게 접근할 수 있도록 파일명은 주로 문제 번호 기준으로 관리합니다.

## Contents

- `class_3/`, `class_4/`: BOJ solved.ac 클래스별 문제 풀이
- `binary_search/`: 이분 탐색 유형 풀이
- `shortest_path/`: 최단 경로 유형 풀이
- `programmers/`: 프로그래머스 문제 풀이
- `algorithm_notes.md`: 알고리즘 개념과 코드 템플릿 정리
- `python_cheatsheet.md`: 코딩 테스트용 파이썬 문법과 표준 라이브러리 정리

## Notes

자주 쓰는 알고리즘 템플릿은 [algorithm_notes.md](algorithm_notes.md)에 모아두었습니다. BFS, DFS, 다익스트라, 플로이드 워셜, 위상 정렬, Union-Find, MST, 이분 탐색, 투 포인터, 슬라이딩 윈도우, DP, 문자열 알고리즘, 수학 관련 패턴을 포함합니다.

파이썬 문법과 표준 라이브러리 사용법은 [python_cheatsheet.md](python_cheatsheet.md)에 정리했습니다. 빠른 입력, 정렬, `deque`, `heapq`, `bisect`, `itertools`, 문자열 처리, 리스트 초기화, 비트 연산 등을 빠르게 확인할 수 있습니다.

## Rules

- 모든 풀이는 특별한 이유가 없으면 Python으로 작성합니다.
- 문제를 풀 때는 입력 크기, 시간 복잡도, 인덱스 기준을 먼저 확인합니다.
- 다시 볼 만한 풀이 패턴은 노트에 템플릿으로 정리합니다.
